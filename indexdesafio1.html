As tags h1 até h6

As tags h1 até h6 serve como titulos inicias,que são colocadas acima do texto,a tag h1 é recomendada a utilização uma vez, já as demais pode ser utilizadas conforme as necessidades.

Tag hr

(voltar)

A tag hr é utilizada para deixar um traço em baixo do texto como fosse uma divisão de contéudo.

Tag sub

(voltar)

A tag sub define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H2O.

Tag sup

(voltar)

A tag sup define o texto sobrescrito.O texto sobre escrito aparece meio caractere acima da linha normal e, às vezes é renderizaddo em um fonte menor.

Tag blockquote

(voltar)

A tag blockquote especifica uma seção que é citada de outra fonte.

Tag font

(voltar)

A tag font foi usada no HTML 4 para especificar a face da fonte, o tamanho da fonte e a cor do texto.

Tag del

(voltar)

A tag del define o texto que foi excluído de um documento. Os navegadores geralmente riscam uma linha através do texto excluído.

Tag abbr

(voltar)

A tag abbr define uma abreviação ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM."
